Jeu de Devinette - Exercice 2
Description
Petit jeu de console développé en Java où l'utilisateur doit trouver un nombre aléatoire entre 1 et 100.

Règles du jeu
Le nombre est généré aléatoirement.
L'utilisateur a 10 tentatives maximum.
Le programme indique si le nombre secret est plus grand ou plus petit après chaque essai.
Installation
Importer le projet dans IntelliJ et exécuter la classe Main.
